# Robotrace Shortcut Path Planner

2025年全日本ロボトレースコースへ複数のショートカット経路生成方式を適用し、簡易速度計画による予測走行時間を比較する最小構成の検討環境です。

```powershell
$env:PYTHONPATH="src"
python -m robotrace_shortcut_lab
```

結果は`outputs/result.png`へ1枚だけ生成します。CSV、外部ログ、GUI、ファームウェア組み込みは扱いません。

## 比較方式

1. 入力点列そのままの原コース
2. LN5.xファームウェアを参考にしたElastic Band相当
3. 原コース法線方向のFrenet横オフセットだけを更新し、8候補から予測時間で選ぶ方式

新方式は、経路短縮、曲率平滑化、曲率変化抑制、元ライン復元、75 mmオフセット射影、端部保護、100 mm最小半径復元を固定回数の局所更新で行います。候補は1本ずつ評価し、最良パラメータだけでもう一度生成するため、8本分の経路メモリは不要です。

## 仮の比較条件

- 最大速度: 8.0 m/s
- 最低速度: 1.0 m/s
- 加速度: 10.0 m/s²
- 減速度: 30.0 m/s²
- 最大角速度: 1500 deg/s
- GFCP基準速度: 3.0 m/s（半径100 mm）
- GFCP指数: 0.33
- 横オフセット上限: 75 mm
- 最小半径: 100 mm

これらは経路比較用の仮設定であり、実機性能や走行可能性を保証しません。

コースデータの出典とライセンスは`data/courses/README.md`と`data/courses/LICENSE.robotrace_course_cad`を参照してください。
