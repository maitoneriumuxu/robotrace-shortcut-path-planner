# LN5床面投影外形

`LN5_footprint.json`は、競技合法性判定に使うLN5実車の床面投影外形です。

- 座標系は`+X=前方`、`+Y=左方`とする。
- `origin_definition`へ、制御上の経路位置と外形原点の対応を記録する。
- 凸・単一外形は`polygon_vertices_mm`へ反時計回りの頂点列を入れる。
- 非凸または分離部品を含む場合は`polygon_components_mm`へ複数の多角形を入れる。
- タイヤ、基板、モータ、吸引機構など、床面投影へ現れる全要素を含める。
- `front_mm`、`rear_mm`、`left_mm`、`right_mm`へ原点からの確定外寸を記録する。
- CAD、図面、または実測記録を`source`へ明記する。
- 原点、頂点列、出典を照合した後だけ`confirmed`を`true`にする。

現在のファイルは、読み取り専用LN5リポジトリと関連CADを調査しても確定外形を取得できなかったため、頂点列なし・`confirmed=false`です。この状態では大域経路を探索・採用せず、現在の4.471秒Frenet経路へフォールバックします。センサ位置やトレッドから車体外形を推測してはいけません。

`safety_margin_mm`は出典のある保守値だけを設定します。物理的に存在しない拡張領域を「白線へ接触した車体」と数える用途には使いません。
